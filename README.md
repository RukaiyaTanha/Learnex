# 🚀 Learnex

**Smart Classroom Management | Student Performance Tracking | AI Insights**

**Learnex** is a cutting-edge Django web application that helps educators and students **track, analyze, and improve learning outcomes**.  
📊 Real-time analytics for quizzes and courses  
🤖 AI-powered performance predictions  
🖥️ Interactive and intuitive dashboard  
🛠️ Easy course, student, and quiz management
